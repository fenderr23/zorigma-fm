# Zorigma FM

Интерактивный сайт-поздравление в виде музыкального плеера.

## Описание

Zorigma FM - это статический веб-сайт, который представляет собой персонализированное поздравление для подруги в виде музыкального плеера с двумя режимами:
- **Режим плейлиста**: треки-воспоминания с историями и медиа
- **Режим генератора комплиментов**: случайные комплименты при нажатии кнопки

## Особенности

- 🎵 Стильный дизайн в эстетике музыкального плеера
- 📱 Полная адаптивность для мобильных устройств
- 🎨 Настраиваемая цветовая схема и шрифты
- 📝 Режим редактирования для настройки контента
- 📤 Экспорт данных в JSON файл
- 🚀 Статический сайт - можно разместить на любом хостинге

## Структура проекта

```
zorigma-fm/
├── src/                    # Исходный код
│   ├── components/        # React компоненты
```

## 🚀 Быстрый старт

### Установка и запуск

1. **Установите зависимости:**
```bash
cd zorigma-fm
npm install
```

2. **Запустите в режиме разработки:**
```bash
npm run dev
```

3. **Откройте в браузере:** [http://localhost:3000](http://localhost:3000)

### Альтернативный запуск (одной командой)

```bash
./start.sh
```

### 📱 Просмотр сайта

После запуска откройте браузер и перейдите по адресу:
- **Локально:** http://localhost:3000
- **На телефоне:** используйте IP вашего компьютера (например, http://192.168.1.100:3000)

### 📦 Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`. Их можно загрузить на любой хостинг:
- GitHub Pages
- Vercel
- Netlify
- Любой веб-сервер

## Настройка поздравления

### Режим редактирования

1. Откройте сайт в режиме разработки
2. Перейдите в режим редактирования
3. Настройте:
   - Имя подруги
   - Заголовок поздравления
   - Цветовую схему и шрифты
   - Треки-воспоминания (минимум 6)
   - Комплименты (минимум 20)

### Экспорт данных

После настройки нажмите "Экспорт" чтобы сохранить данные в файл `greeting.json`.

### Размещение на хостинге

1. Соберите проект: `npm run build`
2. Загрузите содержимое папки `dist/` на любой статический хостинг:
   - GitHub Pages
   - Vercel
   - Netlify
   - Любой веб-сервер

## Технологии

- **React 18** - библиотека для пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Vite** - сборщик и dev-сервер
- **Framer Motion** - анимации
- **Jest + fast-check** - тестирование (unit + property-based)
- **ESLint + Prettier** - линтинг и форматирование
